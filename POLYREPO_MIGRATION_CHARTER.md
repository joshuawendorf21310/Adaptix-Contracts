This repository is executing a controlled migration from monorepo to polyrepo.

- adaptix-platform-infra
- adaptix-contracts
- adaptix-core
- adaptix-cad
- adaptix-field

Governing law:
Core owns truth. Modules own experience. Interop owns translation. Insight owns interpretation. AI owns augmentation.

Program goals:
- explicit ownership boundaries
- contract-governed integrations
- independent CI/CD per repository
- rollback-safe cutovers
